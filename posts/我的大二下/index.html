大二下存档
#

前序
#

大二已经过完了,现在也已经是暑假的一半了.我为了浪费时间,就来写一篇博客,来记录一下大二下的生活,以此留下一份回忆😊

大二下
#

碎碎念
#

大二下的课程怎么说呢🤔

只有一门课程让我印象深刻,并且也留下小许遗憾😢

CINTA这门课程,就是让我们简单入门了数论以及群论.这门课程的老师是熟悉的斌头,这位老师是我入学这么久以来认为唯一的一位满肚子墨水的老师(接触时间相比ldd长,所以只能暂时遮掩判断)

这门课的内容,说简单也简单,说难也不是很难.我也算是有认真的预习一点然后课上也有认真听课,课后也有认真的总结和做作业.

但是在我自认为能够胸有成竹地拿下期末考试的最高分的时候(因为做的时候都感觉全是做过的,也一个小时多一点就全部做完了,也检查了两三遍) 我却意想不到地落败了,我没有拿下我自认为的理想分数.最让人可笑的是: 在还没有出完分数前,我已经自认为地觉得自己就是那个拿下最高分书的人,还深信不疑,真是可笑🤣

不过除开这一点,这门课还是让我收获颇丰.虽然没有达到理想结果就是了,但是终究结局还算可以

吐槽
#

除了这一门课程也还有一门课程让我在期末周也算是破防了吧,居然能够让我没有把握去应对一门考试,这一门就是逆天的byyl.

不过不幸中的万幸是我们的老师不是牛魔的那个男人:hyl~

我们的老师是msj,这位老师是新来的,也没有这门课程的经验.因此,这门课程能够赢下来的概率还算是高了一点点,但是现在分数也还没有出来,因此我也拿不准,只能说听天由命啦!😮‍💨

假期
#

清明假期
#

这次的清明假期,我本来是预算早一个下午回去的话,我可以避免大塞车.结果我还是无法避免塞车,怎么说呢,车其实是没有塞的.但是一些逆天尼玛司机,开车都不会开,开的慢吞吞导致后面的车塞住,这样的司机我只能说尼玛不会开就别开

回来的时候也还是没有出乎意料地塞车了,喜提两次痛苦坐牢.

五一假期
#

这次的五一假期跟着pgx去了深圳玩了一下,其实也没有去哪里玩,只能说去了深圳湾公园逛了一下. 并且这次得了口轻溃疡,我真的是哭死😭,没有很好地迟到好吃的,比如:肠粉,鸡脚

这些都还好,最痛苦的是晚上没有睡好,我果然是顶不住呼噜声.不过我也认识到了一个道理:分房睡这个才是版本答案

暑假
#

月中总结
#

暑假也算是过了一半了,计划是有的,执行力度吧,真的难评.

  • 学车,拿下驾照(还差科三科四)
  • OS的学习,尤其是xv6,还有个新增的raspberry rust os
  • abstract algebra的学习
  • CINTA的全部章节的学习
  • 日语的学习
  • 离散数学的学习

从上面的任务列表的完成度可以看出,我的暑假真的是又是没有完全充实呢😮‍💨

不过,我还算是收获了点:

  1. 一个系统的开发流程是怎么样的
  2. 系统的一个项目结构是怎么样的

小声bb:这个暑假我也没有看很多番啊😭,为什么时间就这样过去了どうして

无语时刻
#

我的服务器在前天炸了,就是因为不小心碰了一下外接硬盘的数据线,可能就导致分区挂载失败,又在进行数据传输导致直接炸了. 然后,我的windows下又不小心在bitcomet在删除任务列表的时候选择了删除任务以及文件导致我下载的番全部没了,真的是卧槽泥马😡

我也在这次的事当中明白了:

省钱不省心,两台不合适的设备就不要搞集群,只能说这样好的设备只会被拖垮,也增加风险

最后,我也只能重新下载我的番剧,重新整理我的设备和服务器,让其实现分离. 最终,nuc小主机就是主要是用来存储视频文件的,用来当一个番剧存档点.那台旧电脑就拿来当alist一个自用网盘算了

总结
#

大二就这样结束了,方向也决定了的,就是尽量日本留学,往OS方向发展,但是执行力这一块还是得好好提升啊🙏
