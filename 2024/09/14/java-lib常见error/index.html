java_libеёёи§Ғerror

[!NOTE] еңЁ javac зј–иҜ‘е’ҢдҪҝз”Ё java иҝҗиЎҢж—¶иҮӘеҠЁж·»еҠ еә“пјҢдҪҝз”Ё--module-path ж·»еҠ еә“и·Ҝеҫ„пјҢдҪҝз”Ё--module-add ж·»еҠ еә“

еңЁиҫғж–°зҡ„ Ubuntu зүҲжң¬пјҲеҰӮ 22.04пјүдёӯпјҢopenjdk-8-jdk еҢ…д»…еҢ…еҗ« OpenJDK 8 зҡ„ж ёеҝғеҠҹиғҪпјҢдёҚеҶҚеҢ…жӢ¬ JavaFX еә“гҖӮJavaFX иҮӘ Java 11 иө·иў«з§»йҷӨеҮәдәҶ JDKпјҢжүҖд»ҘеңЁ OpenJDK 8 д№ҹжІЎжңүиҮӘеҠЁеҢ…еҗ« JavaFXгҖӮ

еҺҹеӣ пјҡ

  1. JavaFX зҡ„еҲҶзҰ»пјҡ
    • иҮӘ Java 11 иө·пјҢJavaFX иў«д»Һж ҮеҮҶ JDK дёӯз§»йҷӨпјҢжҲҗдёәдёҖдёӘзӢ¬з«Ӣзҡ„жЁЎеқ—гҖӮOpenJDK 8 е№¶дёҚеҢ…еҗ« JavaFXпјҢеӣ дёәе®ғзҡ„и®ҫи®ЎзҗҶеҝөе’ҢжңӘжқҘзүҲжң¬зҡ„ JavaFX жЁЎеқ—еҢ–зӯ–з•ҘжңүжүҖдёҚеҗҢгҖӮ
  2. JavaFX дёҺ OpenJDK 8пјҡ
    • еңЁ OpenJDK 8 ж—¶пјҢJavaFX жҳҜдҪңдёә JDK зҡ„дёҖйғЁеҲҶжҸҗдҫӣзҡ„пјҢдҪҶеңЁжӣҙй«ҳзүҲжң¬дёӯпјҲеҰӮ Java 11 е’Ңд№ӢеҗҺпјүпјҢе®ғиў«з§»йҷӨеҲ°дёҖдёӘзӢ¬з«Ӣзҡ„еә“дёӯгҖӮеӣ жӯӨпјҢOpenJDK 8 зҡ„еҢ…дёӯдёҚеҶҚеҢ…еҗ« JavaFXгҖӮ

и§ЈеҶіж–№жі•пјҡ

иҰҒеңЁдҪҝз”Ё OpenJDK 8 зҡ„зі»з»ҹдёҠж·»еҠ  JavaFXпјҢдҪ йңҖиҰҒжүӢеҠЁе®үиЈ… JavaFX SDKгҖӮиҜ·жҢүз…§д»ҘдёӢжӯҘйӘӨж“ҚдҪңпјҡ

  1. дёӢиҪҪ JavaFX SDKпјҡ
    • и®ҝй—® OpenJFXе®ҳзҪ‘ дёӢиҪҪдёҺдҪ зҡ„ж“ҚдҪңзі»з»ҹеҢ№й…Қзҡ„ JavaFX SDKгҖӮ
  2. и§ЈеҺӢе№¶й…ҚзҪ® JavaFXпјҡ
    • и§ЈеҺӢдёӢиҪҪзҡ„ SDK ж–Үд»¶гҖӮдҪ дјҡеҫ—еҲ°дёҖдёӘеҢ…еҗ« JavaFX JAR ж–Үд»¶зҡ„ lib ж–Үд»¶еӨ№гҖӮ
  3. зј–иҜ‘е’ҢиҝҗиЎҢ JavaFX еә”з”ЁзЁӢеәҸпјҡ
    • зј–иҜ‘ JavaFX зЁӢеәҸж—¶пјҢдҪҝз”Ё --module-path еҸӮж•°жқҘжҢҮе®ҡ JavaFX JAR ж–Үд»¶зҡ„дҪҚзҪ®гҖӮ
    • иҝҗиЎҢ JavaFX зЁӢеәҸж—¶пјҢдҪҝз”Ё --module-path е’Ң --add-modules еҸӮж•°жқҘзЎ®дҝқ JavaFX жЁЎеқ—еҸҜз”ЁгҖӮ

зӨәдҫӢпјҡ

еҒҮи®ҫдҪ дёӢиҪҪе№¶и§ЈеҺӢзҡ„ JavaFX SDK дҪҚдәҺ /path/to/javafx-sdkпјҢз„¶еҗҺдҪ еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өзј–иҜ‘е’ҢиҝҗиЎҢ JavaFX зЁӢеәҸпјҡ

иҝҗиЎҢпјҡ
